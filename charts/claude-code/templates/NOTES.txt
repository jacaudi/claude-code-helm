Claude Code has been deployed!

Wait for the pod to be ready:

  kubectl wait --namespace {{ .Release.Namespace }} --for=condition=ready pod -l app.kubernetes.io/name={{ include "claude-code.name" . }},app.kubernetes.io/instance={{ .Release.Name }} --timeout=120s

Open a shell in the pod:

  kubectl exec -it --namespace {{ .Release.Namespace }} deploy/{{ include "claude-code.fullname" . }} -c claude -- sh

Run Claude Code:

  claude

The pod mounts /home/node from a PVC by default, so login state and ~/.claude persist across pod restarts.
